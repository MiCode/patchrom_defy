Automatische Software-Installation

Der Standard-USB-Modus für Verizon-Mobiltelefone (Massenspeicher-Modus) stellt neben einem Massenspeichergerät auch eine CD-ROM-Partition zur Verfügung, die automatisch vom Telefon aus die Installation der Software auf einem Host-Computer startet. Einige Autoradios und Fotokioske bieten keine Unterstützung für diese CD-ROM-Partition und arbeiten möglicherweise nicht korrekt damit zusammen. Zur Umgehung dieses Problems haben wir eine Option vorgesehen, die es ermöglicht, die CD-ROM-Partition beim nächsten Anschließen des Geräts vorübergehend auszublenden, so dass das Autoradio bzw. der Fotokiosk den Massenspeicher Ihres Geräts erkennen kann.

Verwendung:

  • Vergewissern Sie sich, dass Ihr Telefon auf den Massenspeicher-Modus eingestellt ist. Dies ist die Standardeinstellung; das heißt, wenn Sie bisher keine Änderung vorgenommen haben, müsste diese Einstellung aktiviert sein. Wenn Sie die Einstellung überprüfen möchten, schließen Sie das Gerät über USB an, wählen Sie im Benachrichtigungsmenü "USB-Verbindung" und vergewissern Sie sich, dass "Massenspeicher" gewählt ist.

  • Trennen Sie die USB-Verbindung.

  • Rufen Sie "Menü - Speicher - Autom. SW-Install. deaktivieren" auf und wählen Sie diese Option aus.

  • Verbinden Sie Ihr Gerät mit dem Autoradio bzw. Fotokiosk.

Jedes Mal wenn Sie diese Option verwenden möchten, müssen Sie das genannte Menü aufrufen und die Option auswählen, BEVOR Sie die USB-Verbindung herstellen.
