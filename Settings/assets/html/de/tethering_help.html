USB-Tethering

Sie können Ihr Telefon über ein USB-Kabel mit Ihrem Computer verbinden, um die Internetverbindung des Telefons auf Ihrem Computer nutzen zu können.
  • Tethering funktioniert mit Windows Vista, Windows 7 und Linux.

  • Sie können die SD-Karte Ihres Telefons nicht auf Ihrem Computer bereitstellen, solange eine USB-Verbindung besteht.

  • Weitere Informationen (einschließlich USB-Tethering mit anderen Betriebssystemen) finden Sie auf http://www.android.com/tether#usb.

Mobiler WLAN-Hotspot

Sie können Ihr Telefon in einen mobilen WLAN-Hotspot verwandeln, um die Internetverbindung Ihres Telefons auf einem oder mehreren Computern bzw. anderen Geräten nutzen zu können.
  • Wenn Ihr Telefon als WLAN-Hotspot fungiert, können Sie die darauf installierten Anwendungen nicht verwenden, um über die WLAN-Verbindung auf das Internet zuzugreifen.

  • Sie konfigurieren den Hotspot über die WLAN-Hotspot-Einstellungen: Einstellung der Client-Startadresse und des nutzbaren Kanals sowie Einstellung für "Rundsenden" oder "Verbergen" des WLAN-Hotspot.

  • Sie können die verbundenen Clients anzeigen lassen, Clients trennen und die Anzahl der Clients beschränken, mit denen eine Verbindung aufgebaut werden kann. Außerdem können Sie per Geräteverwaltung festlegen, welche Clients sich mit dem WLAN-Hotspot verbinden können.

  • Sie können die Verzögerungszeit einstellen, nach der sich der WLAN-Hotspot automatisch abschalten soll, wenn keine Datenaktivität durch die Clients besteht.

  • Weitere Informationen erhalten Sie auf http://www.android.com/tether#wifi.
