Automatic software install

The default USB mode (mass storage) for Verizon phones contains a mass storage device as well as a CD-ROM partition that auto-launches software installation from the phone onto a host computer. Some car stereos and photo kiosks do not support the CD-ROM partition and may not work. To resolve this, we have created this option which will allow a non-persistent change so that the next time you connect the device, the CD-ROM partition will be removed and will allow the car stereo or photo kiosk to see your device's mass storage.

How to use:

  • Make sure your mobile is set to mass storage mode. This is the default selection so, if you have not changed it, it should be set to that now. If you want to check what it is set to, connect the device to USB and, in the notification pull-down, select USB connection and ensure that mass storage is selected.

  • Disconnect from USB.

  • Go to Menu, Storage, Disable SW auto-install and select it.

  • Connect your device to the car stereo or photo kiosk.

Every time you want to use this option, you will need to go into the menu and select it before you connect USB.
