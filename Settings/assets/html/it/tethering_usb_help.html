Tethering USB

È possibile eseguire il tethering tra telefono e computer collegando il telefono con un cavo USB per condividere la connessione Internet del telefono con il computer.
  • Il tethering funziona con Windows Vista, Windows 7 e Linux.

  • Non è possibile montare la scheda SD del telefono sul computer durante il tethering USB.

  • Per ulteriori informazioni, incluso l'uso del tethering USB con altri sistemi operativi, visita l'indirizzo http://www.android.com/tether#usb.
