Partage de connexion via USB

Vous pouvez partager la connexion Internet de votre téléphone portable avec votre ordinateur en reliant les deux appareils à l'aide d'un câble USB.
  • Cette méthode peut être utilisée avec Windows Vista, Windows 7 et Linux.
  • Lorsque vous partagez une connexion via USB, vous ne pouvez pas activer la carte SD du téléphone sur l'ordinateur.
  • Pour plus d'informations, notamment sur le partage de connexion via USB avec d'autres systèmes d'exploitation, consultez la page http://www.android.com/tether#usb.
