Anclaje a red USB

Puedes anclar tu teléfono a tu computadora con un cable USB para compartir la conexión a Internet de tu teléfono con tu computadora.
  • El anclaje funciona con Windows Vista, Windows 7 y Linux.

  • No puedes montar la tarjeta SD de tu teléfono en tu computadora cuando el USB esté anclado.

  • Para obtener más información, incluido el anclaje a red USB con otros SO, consulta http://www.android.com/tether#usb

Punto Wi-Fi móvil

Puedes convertir tu teléfono en un punto Wi-Fi móvil para compartir la conexión a internet de tu teléfono con uno o más computadores u otros dispositivos
  • Cuando tu teléfono actúa como punto de acceso, no puedes usar las aplicaciones de tu teléfono para acceder a Internet a través de la conexión Wi-Fi.

  • Puedes configurar el punto de acceso en Configuración Wi-Fi que incluye: configuración de dirección del cliente de inicio, el canal que puede ser usado y configurar el punto Wi-Fi móvil como visible u ocultarlo.

  • Puedes ver los clientes conectados, los desconectados, y limitar el número de usuarios que pueden conectarse. También puedes especificar qué clientes pueden conectarse al punto Wi-Fi móvil usando dispositivos de administración.

  • Puedes ajustar la duración y desactivar el punto móvil si no hay actividad de datos de los clientes.

  • Para obtener más información, visita http://www.android.com/tether#wifi
