Associação USB

Pode associar o seu telefone ao computador com um cabo USB para partilhar com o computador a ligação à Internet do telefone
  • A associação funciona com o Windows Vista, Windows 7 e Linux

  • Durante a associação USB, não é possível instalar o cartão SD do telefone no computador

  • Para mais informações, incluindo como utilizar a associação USB com outros sistemas operativos, consulte http://www.android.com/tether#usb

Ponto de acesso Wi-Fi móvel

Pode transformar o seu telefone num ponto de acesso Wi-Fi móvel para partilhar a ligação à Internet do telefone com um ou mais computadores ou outros dispositivos.
  • Quando o telefone estiver a ser usado como ponto de acesso, não poderá utilizar as aplicações do telefone para aceder à Internet através da sua ligação Wi-Fi

  • Pode configurar o ponto de acesso Wi-Fi com as definiões de Wi-Fi as quais incluem: defininir o endereço inicial de cliente, o canal que pode ser utilizado e definir o ponto de acesso Wi-Fi como ponto de difusão ou ponto oculto.

  • Pode ver os clientes ligados, os clientes desligados e limitar o número de clientes que podem ligar. Pode também especificar quais os clientes que podem ligar ao ponto de acesso Wi-Fi utilizando dispositivos de gestão.

  • Pode também ajustar o atraso para desligar automaticamente o ponto de acesso Wi-Fi se não houver actividade de dados por parte dos clientes.

  • Para mais informações, visite http://www.android.com/tether#wifi
