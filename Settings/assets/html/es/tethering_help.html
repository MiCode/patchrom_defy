Anclaje de USB

Puedes anclar el teléfono a tu ordenador con un cable USB para compartir la conexión a Internet del teléfono con el equipo.
  • El anclaje funciona con Windows Vista, Windows 7 y Linux.

  • No se puede montar la tarjeta SD del teléfono en el equipo una vez realizado el anclaje de USB.

  • Para obtener más información, incluido el uso del anclaje de USB con otros sistemas operativos, consulta la página http://www.android.com/tether#usb.

Zona Wi-Fi móvil

Puedes convertir tu teléfono en una zona Wi-Fi móvil para compartir la conexión a Internet del teléfono con uno o con varios equipos o dispositivos.
  • Cuando el teléfono actúe como un punto de acceso, no podrás utilizar sus aplicaciones para acceder a Internet a través de la conexión Wi-Fi del dispositivo.

  • Para configurar el punto de acceso, ve a los ajustes de zona Wi-Fi móvil del teléfono. Podrás definir la dirección de inicio del cliente, el canal que se va a utilizar y la visibilidad de la zona Wi-Fi (visible u oculta).

  • Puedes ver qué clientes están conectados, desconectar clientes y limitar el número de clientes que se pueden conectar. Asimismo, puedes utilizar la opción "Administrar dispositivos" para especificar qué clientes se conectan a la zona Wi-Fi.

  • Puedes definir el tiempo de retardo a partir del cual se desactivará de forma automática la zona Wi-Fi cuando no se exista actividad de datos de clientes.

  • Para obtener más información, consulta la página http://www.android.com/tether#wifi.
